자주 묻는 질문 (FAQ)

Q: 습관을 3개 이상 추가할 수 없나요?

MonoGrid는 "집중"을 핵심 가치로 설계되었습니다. 연구에 따르면 한 번에 3개 이하의 습관에 집중할 때 성공률이 가장 높습니다. 핵심 습관만 관리하여 지속 가능한 변화를 만들어보세요.

Q: 데이터가 다른 기기와 동기화되지 않아요

다음을 확인해주세요:

  • 설정 > Apple ID > iCloud에서 앱이 활성화되어 있는지 확인
  • 두 기기 모두 동일한 Apple ID로 로그인되어 있는지 확인
  • 인터넷 연결 상태 확인
  • 앱을 완전히 종료 후 다시 실행

Q: 위젯이 업데이트되지 않아요

iOS는 배터리 절약을 위해 위젯 업데이트 빈도를 제한합니다. 위젯을 탭하여 앱을 실행하면 즉시 업데이트됩니다. 또한 앱 설정에서 "위젯 새로고침" 옵션을 확인해주세요.

Q: 과거 날짜의 기록을 수정할 수 있나요?

네, 그리드에서 원하는 날짜를 탭하면 해당 날짜의 완료 상태를 변경할 수 있습니다.

Q: 데이터를 백업하려면 어떻게 하나요?

MonoGrid는 iCloud를 통해 자동으로 백업됩니다. 별도의 백업 작업은 필요하지 않습니다. iCloud가 활성화되어 있는지 확인해주세요.

Q: 앱을 삭제하면 데이터가 사라지나요?

iCloud 동기화가 활성화되어 있다면 데이터는 iCloud에 보관됩니다. 앱을 다시 설치하면 데이터가 복원됩니다. 완전히 삭제하려면 설정 > Apple ID > iCloud > 저장 공간 관리에서 앱 데이터를 삭제해주세요.

문제 해결 안내

위의 FAQ에서 해결되지 않는 문제가 있다면:

  • 앱을 최신 버전으로 업데이트해주세요
  • 기기를 재시작해보세요
  • 앱을 삭제 후 다시 설치해보세요 (iCloud 동기화 시 데이터 유지)
